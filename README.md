# Introduction
This is a light project for e2e autonomous driving, I tried a project from https://github.com/Vitaee/CarlaAutonomousDriving which is a great project copy of Nvidia e2e Driving.
But speed control and traffic light recognition are not included.
So I try to add this ability to the model and change its structure into a more complex version with 2 classification tasks and 3 regression tasks.
In this project, a well-trained model can control the ego car to drive following the line and stop intime before a red light.
I create this project just for learning and I'm looking for improving. We can have some discussion if you have any fantastic ideas.
